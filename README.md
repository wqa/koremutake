# koremutake

Implementation of Koremutake MeRS encoding/decoding for node.

## Installation
    
    npm install koremutake

## Usage
    
    var koremutake = require('koremutake');
    koremutake.encode(46767000067);
    // output: NZXWIZI=
    koremutake.decode('beluhujegibo');
    //output: beluhujegibo
